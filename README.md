# 🎓 Student Registration Form

A responsive and user-friendly Student Registration Form built using HTML5, CSS3, and JavaScript. This project demonstrates the use of various HTML5 input types, form validation, and modern styling techniques.

## 📌 Project Overview

The Student Registration Form allows users to enter their personal and academic details through a well-designed interface. The form uses HTML5 validation features and JavaScript to ensure accurate user input before submission.

## 🚀 Features

- Responsive and modern UI design
- HTML5 form elements
- Client-side validation
- Required field validation
- Email format validation
- Phone number pattern validation
- Password length validation
- Date picker input
- Number input with range limits
- Radio buttons for gender selection
- Dropdown menu for course selection
- Checkbox for Terms & Conditions
- Success message after registration

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
